		message (line 105)
	
 
	
	

Output given message, with style.


If $message is array or object it will be displayed as a table.  Using @see createTableFromArray() method.  $messageStyleName can be registred style name or array of style options.  If $function is provided and $message type is array it will be  applied to each array element.

    2.

    3.
